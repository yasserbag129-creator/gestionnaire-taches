# Gestionnaire de tâches en Python

Ce projet est un petit **gestionnaire de tâches** écrit en Python.  
Il permet d'ajouter, modifier, supprimer et afficher les tâches dans une liste.  
Chaque tâche a un **texte** et un **état** (faite ou non faite).

## Fonctionnalités

- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Afficher la liste des tâches

## Utilisation

1. Cloner le dépôt ou télécharger le fichier `gestionnaire_taches.py`
2. Ouvrir un terminal et exécuter le script :
   ```bash
   python gestionnaire_taches.py
